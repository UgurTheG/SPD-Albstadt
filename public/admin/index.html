ℹ️ Was ist ein Token?

Ein GitHub Personal Access Token ist ein Passwort-Ersatz, das den Zugang zum Daten-Repository ermöglicht.

So erstellen Sie eines:

  1. Gehen Sie auf github.com/settings/tokens/new
  2. Bei „Note" einen Namen eingeben (z.B. SPD Editor)
  3. Bei „Expiration" eine Laufzeit wählen (z.B. 90 days oder No expiration)
  4. Bei „Select scopes" das Häkchen bei repo setzen (ganz oben)
  5. Unten auf „Generate token" klicken
  6. Das Token (beginnt mit ghp_) kopieren und hier einfügen

Wichtig: Das Token wird nur einmal angezeigt — sofort kopieren!

Das Token wird nur in Ihrem Browser gespeichert und nie an Dritte gesendet.
